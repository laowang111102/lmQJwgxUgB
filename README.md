# 学生成绩管理系统 python230

## 项目概述

学生成绩管理系统是一款使用 Python 开发，旨在帮助学校管理学生成绩的软件系统。它提供了一套完整的解决方案，包括成绩录入、修改、查询以及统计分析等功能。

## 技术栈

- 后端：Python 3.*
- 前端：HTML, CSS, JavaScript
- 数据库：SQLite

## 功能模块

### 核心功能

- 成绩管理
  - 成绩录入
  - 成绩修改
  - 成绩查询
- 学生管理
  - 学生信息录入
  - 学生信息修改
  - 学生信息查询

### 辅助功能

- 成绩统计分析
  - 班级平均分
  - 最高分与最低分
  - 成绩分布

## 系统角色

- 管理员：负责系统的维护与管理，成绩的录入与修改。
- 教师：查询所教授学生的成绩，进行成绩统计分析。
- 学生：查询个人成绩。

## 运行环境

- 操作系统：跨平台
- Python 环境要求：Python 3.*

## 部署步骤

1. 安装 Python 3.*
2. 克隆或下载项目代码
3. 安装依赖库
   ```sh
   pip install -r requirements.txt
   ```
4. 初始化数据库
   ```sh
   python manage.py db_init
   ```
5. 运行应用
   ```sh
   python manage.py runserver
   ```
